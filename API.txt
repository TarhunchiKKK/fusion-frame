Медиа:
* (GET) localhost:3000/media - получение всех медиа
* (GET) localhost:3000/media/{id} - получение 1 медиа
* (GET) localhost:3000/media/search - поиск медиа по ключеввым словам
* (POST) localhost:3000/media/create - добавить 1 медиа
* (POST) localhost:3000/media/createmany - добавить несколько медиа
* (DELETE) localhost:3000/media/{id} - удаление 1 медиа
* (DELETE) localhost:3000/media - удаление нескольких медиа

Альбомы:
* (GET) localhost:3000/albums - получение всех альбомов
* (GET) localhost:3000/albums/{id} - получить 1 альбом
* (GET) localhost:3000/albums/search/{name} - поиск альбомов по названию
* (POST) localhost:3000/albums - создать альбом
* (DELETE) localhost:3000/albums/{id} - удалить альбом

Пути:
* (GET) localhost:3000/paths - получить все пути
* (POST) localhost:3000/paths - добавить путь
* (DELETE) localhost:3000/paths/{id} - удалить путь
